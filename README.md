### Sliver系列组件

| 组件 | 属性 |
| ------ | ------- |
| SliverList<br>SliverGrid | 同时滚动ListView和GridView |
| SliverAppBar | 实现页面头部区域展开、折叠的效果，类似于Android中的CollapsingToolbarLayout |
| SliverPersistentHeader | 当滚动到边缘时根据滚动的距离缩小高度，有点类似SliverAppBar的背景效果 |
| SliverToBoxAdapter | CustomScrollView只能包含sliver系列组件，使用SliverToBoxAdapter包裹普通控件 |
| CustomScrollView | 使用Sliver组件创建自定义滚动效果的滚动组件 |
| NestedScrollView | 在其内部嵌套其他滚动视图的滚动视图，其滚动位置是固有链接的 |
